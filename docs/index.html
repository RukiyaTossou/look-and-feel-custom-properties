Knoppen

Oba gebruikt veel verschillende de knoppen. De standaard knoppen zijn zwart met witte tekst. Alle knoppen hebben een icon er naast.

Code voorbeeld:

< button class="buttons-zwart"> Bekijk aanbod ▶ /buttons >

Je hebt ook een aantal witte knoppen. Deze "vind" knop staat naast de zoek balk en is in dit geval wit.
Dit kan je gebruiken als een zwart knop niet past bij een bepaalde achtergrond kleur.


Als je cookies moet accepteren krijg je de rode knoppen te zien. De hover heeft een donkere rode kleur zorgt.
